     8	Never load disk cache programs after Casmgr.  Disk cache programs reset
     9	some internal values that are needed in unaltered form by Casmgr.  It
    10	can cause Casmgr to corrupt the hard disk.
    11	
    12	Norton Disk Cache
    13	This program can cause the error "SatisFAXtion board not responding"
    14	if write caching is turned on.  Turn off write caching and SatisFAXtion
    15	will work fine.
    16	
    17	PC-Cache
    18	When using PC-Cache with SatisFAXtion, turn off the caching of disk
    19	writes by adding the following option to the PC-Cache command line
    20	in the autoexec.bat file:
    21	
    22	     /WRITE=OFF
    23	
    24	Super PC-Kwik
    25	May lock up if SatisFAXtion is in the process of accessing the hard
    26	disk to save or send a file while another program is being loaded
    27	from the disk.  This happens with versions of SUPER PC-KWIK below 3.1
    28	when loaded into EXPANDED MEMORY only.  The solution is to load
    29	SUPER PC-KWIK with /H- /D- in the command line.  This turns off
    30	background reads of the disk track (for HD & Floppies respectively).
    31	The problem has been fixed in version 3.1 of SUPER PC-KWIK.  
    32	
    33	V-CACHE by Golden Bow Software
    34	This disk caching software is compatible with Casmgr.  However, in
    35	order to prevent lock-up when Casmgr is accessing a diskette, you must
    36	invoke the cache with the /T=0 parameter.
    37	
    38	For other cache programs, consult your manual to determine how to
    39	turn off background disk writes.

    45	Antivirus programs can conflict with the SatisFAXtion software.  If
    46	you have an antivirus program installed and you have problems with the
    47	SatisFAXtion software, unload the antivirus program (or remove its
    48	command lines from the autoexec.bat and/or config.sys files and restart
    49	your computer) and try again.  Contact the manufacturer of the
    50	antivirus program for possible settings that will work with the
    51	SatisFAXtion software.
